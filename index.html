Darkmown

Darkmown is a full-featured CoffeeScript port of Markdown. Adding extra features, normalizing syntax and fixing some bugs present on other js implementations.

Live Demo »

Normalized syntax

Consistency is better.

  • Only use # for titles from <h1> to <h6>
  • No need to <wrap> urls
  • No need to <wrap> emails

See demo »

Extra Features

Nice features Darkmown has:

  • Auto-linked urls
  • Auto-linked emails
  • Format paragraphs the same way you write
  • More to come :)

See demo »

Fixed bugs

Almost every js Markdown parser out there have bugs when mixing normal syntax with features like auto-link.

Darkmown corrects all these bugs.
