# Darkmown live demo

## Syntax consistency

* Titles like `<h1>` to `<h6>` only render with the # syntax
* Urls are auto-linked no need to wrap them like this `<url>`
* The same for emails

## About

**Darkmown** is a full-featured Markdown library written in CoffeeScript.
